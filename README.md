## Tech Stack

- HTML 5
- CSS 3
- Javascript
- React JS

## Features

- Students Details Form
- Show Students Attendance

## Run Locally

Clone the project

```bash
  git clone https://github.com/Manoranjan-Sethi/Latracal-Student-Attedance.git
```

Go to folder

```bash
 cd Latracal-Student-Attedance
```

Install packages

```bash
 npm i
```

Run the app

```bash
  npm start
```
